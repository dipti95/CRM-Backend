CRM Backend

The project represents the backend of a customer relationship management (CRM) web application. As users interact with the app via some user interface, your server will support all of the functionalities: All REST APIs (GET, POST, PATCH, DELETE)

The application handles the following 5 operations for customers in the "database":

  • Getting all customers through a the /customers path
  • Single customer through a /customers/{id} path
  • Creating a customer through a /customers path
  • Updating a customer through a /customers/{id} path
  • Deleting a customer through a /customers/{id} path
